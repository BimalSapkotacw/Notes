📋 How to Host FREE on GitHub Pages

Step 1: Go to github.com → Create free account

Step 2: Create a new repository (name: notes or anything)

Step 3: Upload this index.html file to the repository

Step 4: Go to repository Settings → Pages → Deploy from branch → main

Step 5: Your site will be live at: https://yourusername.github.io/notes/

⚠️ Note: All data is saved in your browser (localStorage). Export backups regularly using the button above!
